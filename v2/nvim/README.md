# nvim-v2

Hand-rolled config using `lazy.nvim` + `nvim-lspconfig` directly, no framework.

## LSP
- gopls
- vtsls
- lua-ls
- json-ls
- tailwindcss-language-server

## Formatters (conform.nvim)
- gofumpt, goimports — Go
- stylua — Lua
- shfmt — shell
- biome — JS/TS
- oxfmt — markdown

## Linters (nvim-lint)
- golangci-lint — Go
- oxlint — JS/TS

## Treesitter
go, ts, tsx, jsx, lua, json, markdown, markdown_inline, bash, yaml, css, html

## Plugins
- lazy.nvim
- nvim-lspconfig
- conform.nvim
- nvim-lint
- nvim-treesitter
- telescope.nvim
- neo-tree.nvim
- mini.starter
- render-markdown.nvim
- vim-dotenv
